Verziókövető rendszer

Initalize repository-re kattintva válasszuk ki a mappánkat. Message részre írjuk be, hogy Init és kattintsunk a pipára. Így a mappa commit-olva van.

Remote Repository hozzáadása

CTRL+SHIFT+P lenyomásával előhozzuk a Parancspalettát, ahova beírjuk hogy Add Remote, ide megadjuk a GitHubon készített Repository URL-jét

Repository készítése

A GitHub profilunkon a Repositories fülre kattintva a zöld New gomb segítségével új repostiory-t hozunk létre. Miután mindent beálítottunk a Create Repository gombra kattintuk és bemásoljuk az URL-jét az előző lépés helyére.

Commit & push

Ha helyesen végrehajtottunk minden lépést, akkor már csak annyi dolgunk maradt, hogy a commit & push-ra kattintva feltoljuk a mappánkat a GitHub gyűjteményünkbe.
