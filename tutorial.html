Git letöltése és telepítése

A Git egy verziókezelő, arra szolgál, hogy fileok (programok, dokumentációk, stb) különböző verzióit kordában tartsa, elkönyvelje, tárolja és megossza.

GitHub fiók létrehozása

A GitHub egy olyan felhőplatform, amely a Gitet használja alapvető technológiaként. Sign Up gombra kattintva készítsünk egy új fiókot az oldalon.

Visual Studio Code letöltése és telepítése

A Visual Studio Code Támogatja a hibakeresőket, valamint beépített Git támogatással rendelkezik, továbbá képes az intelligens kódkiegészítésre az IntelliSense segítségével.
